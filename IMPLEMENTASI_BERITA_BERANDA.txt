## ðŸ”§ Technical Details

### Query Berita
```php
SELECT id, judul, konten, gambar, tanggal_publikasi, penulis 
FROM berita 
WHERE status = 'publikasi' 
ORDER BY tanggal_publikasi DESC 
LIMIT 3
```

### Query Pengumuman
```php
SELECT id, judul, isi, tanggal_publikasi, penulis 
FROM pengumuman 
WHERE status = 'aktif' 
ORDER BY tanggal_publikasi DESC 
LIMIT 2
```

### Date Format
- Indonesia locale
- Format: "28 Januari 2026"
- Timezone: Server default

### Text Truncation
- Strip HTML tags
- Berita: max 120 karakter
- Pengumuman: max 150 karakter
- Add "..." di akhir

### Image Handling
- If gambar exists: tampilkan dari uploads/berita/
- If no gambar: tampilkan placeholder dari placeholder.com
- Responsive: img-fluid
